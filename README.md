# MusicStore

A music store management application built with C# and .NET.

## Overview

MusicStore is a web application for browsing, managing, and purchasing music. Built as a hands-on project to explore ASP.NET MVC patterns, data modeling, and backend development with C#.

## Tech Stack

- **Language:** C#
- **Framework:** ASP.NET MVC / .NET
- **Database:** SQL Server / Entity Framework

## Features

- Browse music catalog by genre and artist
- Product detail pages
- Shopping cart functionality
- Basic order management
